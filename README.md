# Delance Drywall & Painting LLC

A modern, professional marketing website for Delance Drywall & Painting LLC, serving Orlando, FL and Central Florida.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel-ready

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Development

The site runs at `http://localhost:3000` in development mode.
